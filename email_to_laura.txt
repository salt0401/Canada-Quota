A few things to note:

- TRQ vs B1 data: The TRQ sheets track permit quantities issued (administrative), while the B1 sheet tracks actual customs entries (physical imports). These are different measurements and use slightly different quarter boundaries (~5-6 day offset). This is explained in the B1 sheet.

- B1 availability: B1 data for Q4's matching calendar quarter (Apr-Jun) won't appear until April. Until then, the B1 sheet will show "not available."

- HTS code coverage: The B1 import data uses a selected subset of HTS codes for each product category (listed in the "HTS code covered" sheet). The official TRQ product scope may include additional tariff items beyond what we're tracking. Let me know if you'd like to adjust the code coverage.

- Quarterly transitions: When a new quarter starts, the script automatically creates new sheets (e.g., "non-FTA Q1", "FTA Q1"). Previous quarter sheets remain in the workbook as historical records.
